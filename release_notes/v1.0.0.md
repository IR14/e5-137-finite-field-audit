# E-5-137 Finite-Field Audit v1.0.0

First archived release of the finite-field audit repository in the e-5-137 / GF(137) companion series.

This release includes:

- source code for the Python/C++ audit sandbox;
- active five-test reproducibility matrix;
- Zenodo metadata in `.zenodo.json`;
- citation metadata in `CITATION.cff`;
- English LaTeX source and rendered PDF;
- Russian LaTeX source and rendered PDF;
- companion-repository scope notes in README and paper scope sections.

Primary PDFs:

- English: `papers/e5_137_lepton_generation/build_pdf/main.pdf`
- Russian: `papers/e5_137_lepton_generation/build_pdf/main_ru.pdf`

The repository intentionally shares notation and reproducibility conventions with related e-5-137 / GF(137) audit repositories. The archived scope of this release is the finite-field audit, tests, reports, and bilingual preprint artifacts specific to this paper.
